# 测试用例

## 功能测试

| 编号 | 测试场景 | 输入 | 预期结果 |
|---|---|---|---|
| TC-01 | 管理员登录成功 | admin / Admin@123 | 进入管理员主界面 |
| TC-02 | 学生登录成功 | PB23111001 / Stu@123 | 进入学生主界面 |
| TC-03 | 登录失败 | 错误密码 | 提示账号或密码错误 |
| TC-04 | 新增学生 | 填写完整档案 | 学生列表刷新并出现新记录 |
| TC-05 | 重复选课 | 相同学生选择相同课程 | 数据库唯一约束阻止重复记录 |
| TC-06 | 成绩录入 | 平时分 80，期末分 90 | 自动计算总评 86，等级良好 |
| TC-07 | 成绩修改 | 修改期末分 | 触发器写入审计日志，更新时间变化 |
| TC-08 | 学籍异动登记 | 专业变更 | 异动记录新增且学生专业同步更新 |
| TC-09 | 上传图片附件 | 选择 JPG 文件 | 文件复制到附件目录并写入数据库 |
| TC-10 | 删除附件 | 删除已有文件 | 数据库记录与物理文件同步删除 |

## 非功能检查

- 表结构满足 3NF，学生、课程、成绩等数据无明显冗余。
- 外键约束生效，级联删除符合业务需要。
- 关键查询字段具备索引，满足课堂演示性能要求。
- 附件采用路径存储，不直接保存大文件二进制内容。
